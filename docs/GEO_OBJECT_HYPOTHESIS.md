# GEO Object Reconstruction Lab — Object Hypothesis

## Objective

This document defines the reconstruction target of the laboratory.

The object is not assumed.

The object must emerge from the original geometric signatures.

This document only describes the properties that the reconstructed object must satisfy.

---

# Observable Signatures

The original GEO investigation revealed the persistent appearance of:

- pi/4
- 3/4
- sqrt(3/5)

together with:

- stable angular bands
- geometric ring structures
- complementarity
- residual projection

The object must naturally reproduce these signatures.

---

# Property 1 — Conservation

The object must preserve total structure.

Its observable representation may change.

Its internal coherence must remain invariant.

Examples:

- folding
- projection
- deformation

must not destroy the object.

Only its representation changes.

---

# Property 2 — Complementarity

The object contains at least two coupled components.

These components are not independent systems.

They form a single coherent structure.

Observable changes in one component affect the global state.

---

# Property 3 — Coupling Region

The object contains a transition region.

This region mediates interactions between complementary states.

The coupling region must be capable of producing:

- stable bands
- geometric transitions
- residual signatures

---

# Property 4 — Angular Stability

The object possesses preferred angular configurations.

These configurations are expected to generate:

- equilibrium regions
- maximal coherence
- stable coupling

The appearance of pi/4 is treated as an observational signature rather than an imposed condition.

---

# Property 5 — Projection

Observable geometry is not necessarily identical to internal geometry.

The object must allow:

- local perception
- global structure

to differ without contradiction.

This property motivates:

- ant journey thought experiments
- folded support models
- projection geometry

---

# Property 6 — Residual Geometry

Interactions may generate a residual component.

This residual:

- remains part of the global structure;
- does not disappear;
- may occupy a non-observable projection.

The residual must be geometrically coherent.

---

# Property 7 — Circle / Ellipse Transition

The object must allow two limiting states:

## Ideal State

Perfect coupling.

Single geometric focus.

Maximum coherence.

## Distributed State

Partial coupling.

Multiple geometric foci.

Observable residual structure.

The transition between these states is expected to generate several original GEO signatures.

---

# Reconstruction Criterion

A candidate object is considered successful if it reproduces simultaneously:

- ring behavior;
- stable bands;
- angular equilibrium;
- complementarity;
- projection;
- residual geometry;
- circle/ellipse transitions.

without requiring those properties to be imposed independently.

---

# Final Note

This document intentionally avoids naming the object.

The identification of the object is reserved for Experiment 06.

The object must be recognized as a consequence of reconstruction rather than assumed as an axiom.
