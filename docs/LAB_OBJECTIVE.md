# GEO Object Reconstruction Lab — Objective

## Purpose

This laboratory reconstructs the geometric object that gave rise to GEO.

It does not start from the GEO effective law.
It does not start from Hubble.
It does not start from Gravity Dynamics.
It does not start from Vacuum Harmonics.
It does not start from the GEO motor.

It starts from the original discovery path:

1. mass and gravity are not treated as identical readings;
2. a geometric residual appears;
3. the residual organizes into preferred regions;
4. the preferred regions reveal simple invariants;
5. those invariants suggest a single underlying geometric object;
6. GEO emerges later as the effective predictive law of that object.

## Central question

Given only the original geometric signatures:

- pi/4,
- 3/4,
- sqrt(3/5),
- stable band,
- geometric ring,
- complementarity,

can we reconstruct the minimal object capable of generating all of them?

## Working hypothesis

The original GEO signatures are not independent numerical coincidences.

They are traces of a single coupling geometry.

This geometry contains:

- an observable component;
- a complementary component;
- a coupling/link region;
- a stable angular band;
- a transition between perfect and partial coupling;
- a residual projected outside the observable plane.

## Methodological rule

The object must not be imposed.

The laboratory must let the object appear through reconstruction.

For this reason, the term "geometric superconductor" is reserved for the final identification step.

## Exclusions

This laboratory does not use:

- CLASS outputs;
- Hubble tension data;
- NASA data;
- exoplanet validation;
- Gravity Dynamics Lab;
- Vacuum Harmonic Lab;
- GEO motor equations.

Those are treated as later consequences or external validations.
